// מדריך 01 מתוך 03

חיבור ממשק Meta

פתיחת אפליקציה ב-Meta Developers, קבלת מפתח גישה ומזהה מספר הטלפון, וחיבור לחשבון שלכם.

⏱ 15–20 דקות
🎯 מתאים לכל רמה
🔒 נדרש חשבון Meta
💡
לפני שמתחילים: וודאו שיש לכם חשבון פייסבוק פעיל. לא צריך חשבון עסקי מוכן מראש — נפתח אחד במהלך התהליך.
🌐
// 01
פתיחת חשבון Meta Developers

נכנסים לפורטל המפתחים של Meta ויוצרים אפליקציה חדשה. זה בחינם לחלוטין.

1
גשו לכתובת developers.facebook.com והתחברו עם חשבון הפייסבוק שלכם.
2
לחצו על My Apps ← Create App.
3
סוג האפליקציה: בחרו Business. לחצו "הבא".
4
תנו שם לאפליקציה (לדוגמה: MyBusiness WA) ← לחצו Create App.
📱
// 02
הוספת WhatsApp לאפליקציה

מוסיפים את מוצר WhatsApp לאפליקציה ומחברים מספר טלפון עסקי.

1
בדשבורד האפליקציה ← רשימת מוצרים ← WhatsApp ← Set Up.
2
חברו חשבון עסקי — אם אין, בחרו Create New Business.
3
לחצו Add phone number ← הכניסו את מספר הטלפון שרוצים לחבר.
4
⚠️ המספר לא יכול להיות פעיל ב-WhatsApp הרגיל — יש להסיר אותו מהאפליקציה תחילה.
5
אמתו את המספר ב-SMS ← הכניסו את הקוד שקיבלתם.
🔑
// 03
קבלת מזהה המספר ומפתח הגישה

שני הפרטים הנדרשים לחיבור הממשק. שמרו אותם במקום בטוח — תצטרכו אותם בשלב הבא.

1
בתפריט הצדדי ← WhatsApp ← API Setup.
2
מזהה מספר הטלפון — מספר ארוך. העתיקו ושמרו.
3
מפתח גישה ← Generate. העתיקו מיד — לא תוכלו לראות שוב לאחר סגירה.
// מה לשמור
# שמרו בצורה מאובטחת
מזהה מספר טלפון:  123456789012345
מפתח גישה:        EAABsbC...long_token
מזהה WABA:        9876543210
🔐
לסביבת ייצור: צרו משתמש מערכת קבוע דרך Business Settings ← System Users. מפתח הגישה שלו לא פג תוקף.
⚡
// 04
חיבור לחשבון AutoScale
1
נכנסו לדשבורד AutoScale ← הגדרות ← חיבור WhatsApp.
2
הכניסו את מזהה מספר הטלפון ואת מפתח הגישה.
3
לחצו חבר — תראו "מחובר ✓" בירוק לאחר כמה שניות.
🧪
// 05
הודעת בדיקה ראשונה

שלחו הודעת בדיקה ישירות דרך ממשק Meta כדי לאמת שהכל עובד לפני שממשיכים.

// קריאת רשת לבדיקה
POST https://graph.facebook.com/v18.0/{PHONE_ID}/messages
Authorization: Bearer {TOKEN}
Content-Type: application/json

{
  "messaging_product": "whatsapp",
  "to": "972501234567",
  "type": "text",
  "text": { "body": "בדיקה מוצלחת! 🎉" }
}
✅
הצלחה: קיבלתם הודעת WhatsApp מהמספר העסקי שלכם. אם הופיעה שגיאה — בדקו שמפתח הגישה הועתק במלואו.
📌 // סיכום מהיר
מזהה מספר הטלפון
WhatsApp ← API Setup
מפתח גישה
API Setup ← Generate
גרסת ממשק
v18.0
כתובת בסיס
graph.facebook.com

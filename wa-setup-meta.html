חיבור לMeta API

יצירת אפליקציה ב-Meta Developers, קבלת הפרטים הנדרשים, וחיבור לחשבון AutoScale שלכם.

⏱ כ-15–20 דקות
🎯 רמה: בסיסי
🔒 נדרש: חשבון Meta Business
💡
לפני שמתחילים: וודאו שיש לכם חשבון Facebook/Meta רגיל. לא חייבים Business Manager מיידית — נפתח אחד במהלך המדריך.
🌐
שלב 1

פתיחת חשבון Meta Developers

נכנסים לפורטל המפתחים של Meta ומגדירים חשבון מפתח. זה חינמי לחלוטין.

1
גשו לאתר developers.facebook.com והתחברו עם חשבון Facebook האישי שלכם.
2
לחצו על "My Apps" בתפריט העליון, ואז "Create App".
3
כשישאלו מה סוג האפליקציה — בחרו "Business". לחצו Next.
4
תנו שם לאפליקציה (לדוגמה: "MyBusiness WhatsApp"). לחצו "Create App".
⚠️
Meta עשויה לבקש אימות זהות בפעם הראשונה. זה תהליך רגיל — עקבו אחרי ההוראות שלהם.
📱
שלב 2

הוספת מוצר WhatsApp לאפליקציה

אחרי יצירת האפליקציה, מוסיפים את מוצר WhatsApp ומגדירים מספר טלפון.

1
בדשבורד האפליקציה, גללו מטה לרשימת מוצרים. מצאו "WhatsApp" ולחצו "Set Up".
2
Meta תבקש לחבר Business Portfolio (לשעבר Business Manager). אם אין לכם — לחצו "Create New Business" ומלאו את הפרטים.
3
בדף ה-WhatsApp Setup, תראו "Phone Number". לחצו "Add phone number".
4
הכניסו את מספר הטלפון שרוצים לחבר. חשוב: המספר לא יכול להיות כבר פעיל בWhatsApp רגיל. אם הוא כבר WhatsApp — צריך להסיר אותו קודם מהאפליקציה הרגילה.
5
אמתו את המספר ב-SMS או שיחה. הכניסו את הקוד שתקבלו.
🔑
שלב 3

קבלת Phone Number ID ו-Access Token

אלו שני הפרטים שצריכים כדי לחבר ל-AutoScale. תצטרכו אותם בשלב הבא.

1
בתפריט השמאלי, לחצו על WhatsApp → API Setup.
2
תמצאו את Phone Number ID — מספר ארוך שנראה כמו: 123456789012345. העתיקו אותו ושמרו.
3
ליד Access Token, לחצו "Generate" (או עברו ל-System User לטוקן קבוע — מומלץ לסביבת ייצור).
4
העתיקו את ה-Token המלא ושמרו אותו במקום בטוח. לא תוכלו לראות אותו שוב אחרי סגירת הדף.
הפרטים שצריך לשמור 
# שמרו את הפרטים האלו בצורה מאובטחת

Phone Number ID:  123456789012345   # דוגמה — יהיה שלכם
Access Token:     EAABsbC...ABCD    # טוקן ארוך מאוד
WhatsApp Business Account ID: 9876543210
🔐
טיפ אבטחה: לסביבת ייצור, צרו System User קבוע דרך Business Settings ← Users ← System Users. הטוקן שלו לא פג תוקף ולא תלוי בחשבון האישי שלכם.
⚡
שלב 4

חיבור לחשבון AutoScale

עכשיו מכניסים את הפרטים שאספנו לתוך מערכת AutoScale.

1
נכנסו לדשבורד AutoScale שלכם.
2
לחצו על הגדרות → WhatsApp Integration.
3
הכניסו את ה-Phone Number ID ו-Access Token שקיבלתם מ-Meta.
4
לחצו "חבר" — המערכת תבדוק את התקינות ותאשר חיבור מוצלח.
✅
הצלחה נראית כך: המערכת תציג "Connected — מחובר" בירוק. אם תקבלו שגיאה — בדקו שהטוקן הועתק בשלמותו ושה-Phone Number ID נכון.
🧪
שלב 5

שליחת הודעת בדיקה

נשלח הודעה ראשונה לוודא שהכל עובד.

1
בדשבורד, לחצו על Test Message.
2
הכניסו את מספר הטלפון שלכם (עם קידומת מדינה: 972 לישראל).
3
לחצו שלח. תוך כמה שניות תקבלו הודעת WhatsApp מהמספר העסקי שלכם.
דוגמה — API Test Request 
# שליחת הודעת בדיקה ישירות ל-Meta API (HTTP)
POST https://graph.facebook.com/v18.0/{PHONE_NUMBER_ID}/messages

Headers:
  Authorization: Bearer {ACCESS_TOKEN}
  Content-Type: application/json

Body:
{
  "messaging_product": "whatsapp",
  "to": "972501234567",
  "type": "text",
  "text": { "body": "שלום! הבדיקה עבדה 🎉" }
}
📌 סיכום — מה צריך מ-Meta
Phone Number ID
מ-API Setup → Phone Numbers
Access Token
מ-API Setup → Generate Token
API Version
v18.0 (עדכני)
Base URL
graph.facebook.com

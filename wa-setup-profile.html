שלב 1

כניסה ל-WhatsApp Manager

1
גשו ל-business.facebook.com והתחברו.
2
בתפריט השמאלי (או ב-Business Settings), חפשו WhatsApp Manager.
3
בחרו את ה-WhatsApp Business Account שיצרתם במדריך 1.
✏️
שלב 2

עריכת פרטי הפרופיל

בחרו את מספר הטלפון שחיברתם, ולחצו על Profile (פרופיל).

1
בחרו את מספר הטלפון מהרשימה ← לחצו Settings ← Profile.
2
לחצו Edit (עריכה) לעדכון הפרטים.
שדה מה להכניס חשיבות
Display Name (שם תצוגה) שם העסק המלא — כפי שהלקוחות יראו אותו חובה
Profile Photo לוגו של העסק — ריבועי, 640×640 px, JPG/PNG מומלץ מאוד
Category (קטגוריה) בחרו את הקרובה ביותר לעסק שלכם (שירותים, קמעונאות, וכו') חובה
Description (תיאור) עד 256 תווים — מה העסק עושה בקצרה מומלץ
Address (כתובת) כתובת העסק הפיזי (אם יש) אופציונלי
Email אימייל צור קשר של העסק אופציונלי
Website אתר האינטרנט של העסק אופציונלי
✅
שלב 3

בקשת אימות Display Name

שם התצוגה דורש אישור של Meta לפני שיופיע ללקוחות.

1
אחרי שמירת הפרטים, ב-Phone Numbers תראו את השם עם סטטוס "Pending" (ממתין לאישור).
2
אישור Meta לוקח בדרך כלל עד 24 שעות. לפעמים כמה דקות.
3
כשמאושר, הסטטוס ישתנה ל-"Approved" והשם יופיע ללקוחות.
⚠️
שם תצוגה חייב להיות שם העסק האמיתי. Meta דוחה שמות גנריים, שמות של מוצרים בלבד, או שמות שאינם קשורים לעסק.
📋
שלב 4 (אופציונלי אך מומלץ)

יצירת Message Template ראשון

Templates הן הודעות מוכנות מראש שאפשר לשלוח ליוזמה (outbound). לדוגמה — הודעת ברוכים הבאים, תזכורת פגישה, וכו'.

1
ב-WhatsApp Manager ← Message Templates ← Create Template.
2
בחרו קטגוריה: Utility (שימושי) לתזכורות ועדכונים, Marketing לפרומושן.
3
כתבו את תוכן ההודעה. משתנים מסומנים {{1}}, {{2}} וכו' — אלו הם הנתונים הדינמיים שתזינו בעת השליחה.
4
שלחו לאישור. Meta בדרך כלל מאשרת Templates תוך שעה עד 24 שעות.
💡
Template לדוגמה: "שלום {{1}}, תזכורת לפגישתך ב-{{2}} בשעה {{3}}. לאישור ענו 'כן'." — Meta תאשר הודעות ברורות ועסקיות.
